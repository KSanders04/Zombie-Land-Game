# Zombie Land

## Instructions:

- Use keys WASD to move around
- Use a mouse to aim and shoot at zombies
- Use the R key to reload weapon
- You have 100 health
- Shoot as many zombies as you can

## Technology and Languages Used:

- Spyder IDE
- SimpleGE (Game Engine)
- Python
- Pygame Library
